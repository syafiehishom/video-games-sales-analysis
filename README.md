# video-games-sales-analysis
Data analysis project exploring video game sales trends using Python (Pandas, Matplotlib)

## 📊 Objective
This project analyzes video game sales data to identify trends in platforms, genres, and overall market performance.

## 🧹 Data Cleaning
- Removed rows with missing sales data
- Filtered dataset to include valid critic scores
- Converted release_date to datetime and extracted year

## 📈 Key Insights

### 🎮 Platform Performance
- PlayStation 4 is the top-performing console
- PlayStation platforms dominate overall sales

### 📅 Sales Over Time
- Sales peaked around 2014
- Decline observed in later years

### ⭐ Critic Score Impact
- Weak positive correlation (~0.32)
- Reviews influence sales but are not the main driver

## ⚠️ Limitations
- Dataset reduced significantly after cleaning (282 rows)
- Analysis limited to games with complete data

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
